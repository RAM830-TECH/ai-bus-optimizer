# AI Bus Optimizer

Intelligent Transportation System for optimizing public bus networks.

## Features
- Route Optimization
- Demand Prediction
- Traffic Monitoring
- Bus Management
- Real-time Tracking
- Analytics & Reporting

## Tech Stack
- Backend: Java Spring Boot
- Frontend: React 18+
- Database: PostgreSQL + PostGIS
- Docker
